💊MedSimply API

I designed and developed a healthcare medication management system focused on ensuring safe, timely, and accountable medication administration. The system is tailored for clinical and caregiving environments, where accurate tracking and scheduling of medication is critical. It enables authorized users to manage prescriptions, monitor dosage schedules, and maintain detailed audit logs for accountability. Built using Django, the application follows a modular architecture, with separate apps handling patients, medications, scheduling, and administration records. This structure improves scalability, maintainability, and clarity of the system design. Through this project, I strengthened my skills in backend development, system architecture, and building solutions that address real-world healthcare challenges. Tech Stack: Python, Django, MySql, REST APIs, Git
